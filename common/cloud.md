# 클라우드

## 전통적이 배포 방식
- 컴퓨터 한 대에 하나의 OS
  
## 가상화 배포 방식
- 컴퓨터의 하드웨어를 소프트웨어적으로 구현
- 한대의 컴퓨터를 가지고 여러개의 OS를 구동할 수 있게

## 하이퍼바이저
- 하나의 시스템 상에서 가상 컴퓨터를 여러개 구동할 수 있도록 해주는 중간계층
